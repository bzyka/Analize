Analize

Вы — аналитик команды, которая разрабатывает программное обеспечение для автомата, печатающего фото из Instagram.  
На встрече с представителем заказчика выяснили, что автомат должен работать следующим образом:  

При нажатии на кнопку «Выбрать фото для печати» на главном экране пользователю предложены 2 способа — выбрать фото в Instagram или загрузить фото с телефона.  
Для выбора фото из Instagram пользователю необходимо указать свой логин. Профиль пользователя должен быть открытым.  
Для выбора фото с телефона, пользователю нужно соединить телефон с автоматом для печати фото через кабель для зарядки.  
Пользователь может выбрать количество фотографий для печати через экран автомата. Минимальное количество фото — 1, максимальное —100 (если в автомате достаточно бумаги).  
Оплатить печать фото можно только по карте через физический терминал бесконтактной оплаты.  
Печать фото запускается после успешной оплаты выбранного пакета фотографий.  
Если пользователь вводил логин Instagram для печати фотографий, то после завершения печати его страница в автомате должна быть закрыта.  
Если пользователь присоединял телефон для печати фотографий, то после завершения печати автомат будет сигнализировать о необходимости забрать телефон.

1. Формализованные требования к программному обеспечению:  
Функциональное требование (FR1): Система должна предоставлять пользователю возможность выбора источника фотографий для печати: Instagram или телефон.  
Функциональное требование (FR2): Система должна запрашивать логин пользователя Instagram при выборе источника фотографий “Instagram”.  
Нефункциональное требование (NFR1 - Производительность): Система должна проверять логин Instagram пользователя не более чем за 3 секунды.  
Интерфейсное требование (IR1): На экране выбора количества фотографий должен отображаться текущий лимит печати (максимальное количество фотографий, которое можно напечатать, исходя из доступной бумаги).  
Требование к безопасности (SR1): После завершения печати система должна автоматически очищать временные данные: закрывать страницу Instagram или сигнализировать о необходимости забрать телефон.

3. Текстовое описание варианта использования “Выбрать фото из Instagram”:  
Название: Выбрать фото из Instagram  
Действующее лицо: Пользователь  
Цель: Пользователь хочет выбрать фотографии для печати из своего аккаунта Instagram.  
Предусловия:  
Автомат находится в режиме главного экрана.  
Автомат подключен к сети Интернет.  
У пользователя есть аккаунт в Instagram.  
Аккаунт пользователя в Instagram является публичным (открытым).  
Основной поток событий:  
Пользователь нажимает на кнопку “Выбрать фото для печати” на главном экране автомата.  
Автомат отображает варианты выбора источника фотографий: “Печать фото из Instagram” и “Печать фото с телефона”.  
Пользователь выбирает “Печать фото из Instagram”.  
Автомат запрашивает у пользователя логин его аккаунта Instagram.  
Пользователь вводит свой логин Instagram.  
Автомат проверяет существование указанного аккаунта Instagram и его открытость.  
Автомат отображает последние фотографии из аккаунта пользователя Instagram.  
Пользователь выбирает желаемые фотографии для печати.  
Автомат переходит к варианту использования “Выбрать количество фото”.  
Альтернативные потоки событий:  
6а. Логин Instagram не найден:  
6а.1. Автомат отображает сообщение об ошибке “Аккаунт Instagram не найден”.  
6а.2. Автомат предлагает пользователю повторно ввести логин или вернуться к выбору источника фотографий.  
6а.3. Пользователь либо повторяет ввод логина (возврат к шагу 5), либо возвращается к выбору источника фотографий (возврат к шагу 2).  
6б. Аккаунт Instagram пользователя закрыт:  
6б.1. Автомат отображает сообщение об ошибке “Аккаунт Instagram закрыт. Сделайте его открытым, чтобы продолжить.”  
6б.2. Автомат предлагает пользователю вернуться к выбору источника фотографий (возврат к шагу 2).  
7а. Не удалось загрузить фотографии из Instagram:  
7а.1. Автомат отображает сообщение об ошибке “Не удалось загрузить фотографии из Instagram. Проверьте подключение к интернету или повторите попытку позже.”  
7а.2.  Автомат предлагает пользователю повторить попытку загрузки или вернуться к выбору источника фотографий (возврат к шагу 2).  
Постусловия:  
Отображены фотографии из аккаунта пользователя Instagram (если основной поток событий завершен успешно).  
Отображено сообщение об ошибке (если произошла ошибка).  


4. Обработка системных ошибок:  
При реализации программного обеспечения необходимо предусмотреть обработку следующих системных ошибок:  
Ошибка подключения к интернету: Невозможность подключения к Instagram API для получения данных пользователя или фотографий. Решение: отображение информационного сообщения пользователю с предложением проверить подключение и повторить попытку.  
Ошибка API Instagram: Ошибки, возвращаемые Instagram API (например, превышение лимита запросов, временная недоступность сервиса). Решение: отображение информационного сообщения пользователю с предложением повторить попытку позже. Логирование ошибки для последующего анализа.  
Ошибка связи с платежным терминалом: Невозможность установить связь с терминалом для приема оплаты. Решение: отображение информационного сообщения пользователю с предложением обратиться к администратору. Логирование ошибки для последующего анализа.  
Ошибка печати: Замятие бумаги, отсутствие бумаги, ошибка печатающей головки. Решение: Приостановка процесса печати, отображение информационного сообщения пользователю с указанием проблемы. Возможность отмены текущего процесса печати и возврата средств (в случае успешной оплаты).  
Ошибка чтения данных с телефона (при загрузке с телефона): Проблемы с подключением телефона, несовместимость форматов файлов, поврежденные файлы. Решение: Отображение информационного сообщения пользователю с предложением проверить подключение телефона, выбрать другой файл или использовать другой источник фотографий.  
Недостаточно средств на карте пользователя: Отображение соответствующего сообщения на экране автомата и предоставление пользователю возможности использовать другую карту или другой способ оплаты (если таковой предусмотрен).  
Попытка ввести невалидные данные в логин Instagram: проверка на минимальное количество символов, недопустимые символы и выдача сообщения об ошибке при необходимости.
